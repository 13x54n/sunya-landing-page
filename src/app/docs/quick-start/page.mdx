# Quick Start

Get up and running with Sunya in two steps: create a config and run a scan.

## 1. Create config

Create `sunya.config.json` in your project root:

```json
{
  "contractsDir": "./contracts",
  "tools": [
    {
      "name": "slither",
      "command": "slither",
      "args": [],
      "parser": "slither"
    },
    {
      "name": "mythril",
      "command": "myth",
      "args": ["analyze"],
      "parser": "mythril"
    }
  ],
  "output": {
    "format": "table"
  }
}
```

## 2. Run scan

```bash
npx sunya scan
```
